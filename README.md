# 入力デバイスプラグイン製作環境について

- OSはWindows11をサポート

## 開発環境
- Visual Studio 2022で開発
  .Net Framework 3.5を使用し開発を行う。
  ※Visual Studioインストール時に"個別のコンポーネント"よりインストールが必要
